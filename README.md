# コード勉強
React Nativeを使用してゼロから開発した時間割管理アプリ。
時間割表示の静的な表示から段階的にデータ保存機能などを追加していきました。

##🎩アプリの機能
- **時間割管理**:科目名、タイプの編集
- **課題管理**:課題の追加、削除、期限管理と強調表示
- **データの永続化**:アプリに入力したデータの保持

##🔨使用技術
- **FrameWork**: React Native
- **Language**: JavaScript
- **Storage**: AsyncStorage
- **UI**: Stylesheet, @expo/vector-icons
- **tools**: VScode, GitHub, Gemini
